# vue-snake-ui

>  vue-snake-ui是一套采用 Vue 2.0 作为基础框架实现的组件库


## 安装
```shell
npm install vue-snake-ui -S
```

## 快速开始
``` javascript
import Vue from 'vue'
import Snake from 'vue-snake-ui'

Vue.use(Snake)

```
